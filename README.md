# QuickQuiz — Web Quiz App

A small single-page quiz application you can run in VS Code.

## Features
- Start screen with category & difficulty selection
- Shuffles questions and options each run
- One question at a time with progress bar
- Immediate feedback (correct/incorrect) and final detailed feedback
- Simple, dependency-free (vanilla HTML/CSS/JS)

## How to run
1. Unzip `quiz_app.zip` and open the folder in VS Code.
2. Open `index.html` in Live Server (recommended) or directly in your browser.
3. Choose category & difficulty, then click **Start Quiz**.

## Files
- `index.html` — main UI
- `styles.css` — styles
- `app.js` — quiz logic
- `questions.json` — the question bank you can extend

Feel free to expand the `questions.json` with more questions and categories.
